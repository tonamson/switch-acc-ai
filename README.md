# switch-acc-ai

`switch-acc-ai` là công cụ hỗ trợ chuyển đổi nhanh giữa nhiều tài khoản AI như
Codex, Claude và các dịch vụ tương tự.

Công cụ này giúp tách biệt profile tài khoản trên máy local, tránh phải dùng
hệ thống proxy hoặc cách đổi môi trường dễ dẫn tới vi phạm chính sách của nhà
cung cấp.

## Install

```bash
npm install -g switch-acc-ai
```

## Run

```bash
sacc
```
